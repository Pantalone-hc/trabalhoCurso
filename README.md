# Projeto de Plataforma de Cursos

## Como rodar o projeto:

### Back-end (API)
1. Entre na pasta `api-cursos`.
2. Rode `npm install`.
3. No arquivo `.env` insira a url correta.
4. Rode `npx prisma db push` para subir as tabelas.
5. Rode `npm start`.

### Front-end
1. Entre na pasta `front-bastet`.
2. Rode `npm install`.
3. Rode `npm run dev`.
